# Wordbook
簡単な単語帳アプリです。

***DEMO;***
![demo](https://user-images.githubusercontent.com/51501611/62747745-be2a2200-ba90-11e9-9551-fbd4c1b72d17.gif)

### 主な使用技術
- DB PostgreSQL
- ユーザー管理機能 Devise
- ページネーション機能 kaminari
- 検索・絞りこみ機能 Ransack
- テスト RSpec
- フロント開発　
  - slim
  - bootstrap
- Ruby 2.6.3
- Rails 5.2.3
